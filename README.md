# My Fitness Tracker

Mobile-first React app for daily workout logging with 10 default arm/leg exercises (Curls, Hammer Curls, etc.). Track sets/reps/weight, view week-on-week charts. Firebase-backed, GitHub Pages deploy.

## Quick Start

```bash
npm install
npm run dev
```

## Features
- Daily log with pre-loaded activities
- Week-on-week progress charts (Chart.js)
- Real-time sync (Firebase Firestore)
- Mobile responsive (Tailwind CSS)

## Deploy
```bash
npm run deploy  # After gh-pages setup
```
